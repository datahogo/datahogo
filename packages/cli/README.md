# datahogo

Open-source security scanner that runs on your machine — your code never leaves your laptop.

```bash
npx datahogo
```

300+ checks across JS/TS, Python, Go, Java, PHP, C#, mobile, and Supabase. Plain-language explanations and AI fixes with your own Anthropic API key.

Full docs, options, and CI examples: [github.com/datahogo/datahogo](https://github.com/datahogo/datahogo)
